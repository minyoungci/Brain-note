RESULTS.md에서 SYSU leak=-25%, Otsu leak=-50%로 음수 보고됨. β≈0인 비검출 도구는 뇌실 보정 후 부호만 바뀌어도 공식이 불안정해짐. **naive(β=+0.095, 옳은 방향이 아님)의 leak=6%** 해석도 공식 의미가 불분명 — "위축 누출이 6%"가 아니라 "보정 전후 β 비율"이 6%인데, wrong-sign 검출자엔 이 지표가 맞지 않음.

**3. MICCAI split 컬럼 파싱 — `dissociation_analysis.py:68`**
```python
d["split"] = d.sid.astype(str).str.split("_").str[0]
```
MICCAI 2017 WMH 데이터의 subject ID 포맷이 `{split}_{...}` 구조임을 가정. 실제 파일명 규칙이 다르면 **모든 행이 "train" 또는 "test" 단 하나의 값**만 갖게 되어 test-split-only bias-control이 무의미해짐. `split` 컬럼이 없으면 fallback으로 전체를 포함하는 코드 경로(`d["split"] = d.sid...`)가 있긴 하나, 이 파싱이 맞는지 검증 증거가 diff에 없음. [불확실]

**4. Mann-Whitney p=0.05, n=6 — `RESULTS.md:dissociation 섹션`**
"PERFECT separation, Mann-Whitney p=0.05"인데 도구 수는 검출자 3, 비검출자 3으로 총 n=6. 이 표본 크기에서 Mann-Whitney p=0.05는 최솟값에 가깝고(3×3 순열로 가능한 최소 p ≈ 0.1, 단측 0.05). **"완벽 분리"라는 표현은 맞지만 통계적 검증력이 아니라 완전 분리(perfect separation) 그 자체에서 나온 p임을 명기해야 함**.
