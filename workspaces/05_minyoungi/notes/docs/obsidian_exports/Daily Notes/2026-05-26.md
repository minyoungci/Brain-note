## 3. 오늘 발견한 기술적 문제: fold/variant order-dependent seed

오늘 중요한 구현 이슈를 발견했다.

문제:

```text
fold sweep 순서나 variant 순서가 바뀌면 teacher/student initialization이 달라져서,
selected-fold 결과와 full-fold 결과를 그대로 비교하기 어려울 수 있었다.
```

수정 방향:

```python
def stable_offset(s: str) -> int:
    return sum((i + 1) * ord(ch) for i, ch in enumerate(s))
```

그리고 다음처럼 seed를 fold/variant 기준으로 reset하도록 수정했다.

```text
run_fold:    seed_all(args.seed + stable_offset(heldout))
run_variant: seed_all(args.seed + stable_offset(heldout) + stable_offset(variant))
```

의미:

> 앞으로 full LOCO, selected-fold, variant sweep을 비교할 때 sweep order 때문에 결과가 달라지는 문제를 줄인다.

공부 포인트:

- seed를 한 번만 고정하는 것은 충분하지 않다.
- multi-fold / multi-variant sweep에서는 **실험 단위마다 deterministic seed reset**이 필요하다.
- 특히 작은 의료영상 실험에서는 initialization 차이가 fold별 결론을 바꿀 수 있다.

- CE 0.05: hard-label anchor를 더 약하게 했을 때 NACC가 살아나는지 확인
- CE 0.10: Gate04 기준점 재현

- CE 0.25: 더 강한 CE anchor
- teacher_kl_repeat: CE 없이 Teacher-S만 따르는 기준점



> Teacher-B는 학생 input에 age/sex를 넣는 것이 아니다. 학생은 여전히 T1w voxel만 본다. age/sex는 teacher diagnostic signal로만 쓴다.

해석:

- Teacher-B가 NACC에서 크게 좋아지면, NACC는 순수 ROI 형태보다 demographic/context axis가 강하게 작용하는 fold일 수 있다.
- Teacher-B도 안 좋아지면, NACC는 ROI teacher 자체 또는 데이터/QC/label-distribution 쪽을 더 봐야 한다.

## 7. 오늘의 연구적 결론

### 결론 1 — 이 recipe는 버릴 것이 아니다

`Teacher-S KL + 0.1 CE`는 평균적으로 baseline_06을 넘겼고, 특히 KDRC를 살렸다.

이것은 중요한 신호다.

> ROI-aware teacher distillation은 단순한 handcrafted ROI baseline이나 image-only CNN과 다른 방향의 representation improvement를 만들 수 있다.

### 결론 2 — 하지만 최종 recipe로 고정하기엔 이르다

NACC와 AJU가 남았다.

- AJU: sensitivity / CE anchor tuning 문제일 가능성
- NACC: teacher ceiling / cohort shift / ROI signal limitation 문제일 가능성

즉, 지금의 결론은:

```text
VLM scaling으로 가기 전, objective robustness와 teacher-ceiling diagnosis를 한 번 더 해야 한다.
```

### 결론 3 — NACC 실패도 연구적으로 가치 있다

NACC가 끝까지 해결되지 않는다면 그것도 실패가 아니라 중요한 boundary condition이다.

가능한 claim:

> 모든 cohort에서 ROI morphology-only signal이 동일하게 강하지 않다. NACC처럼 ROI-only Teacher-S ceiling이 낮은 cohort에서는 demographic/context, QC, 또는 spatial ROI-token architecture가 필요할 수 있다.

## 8. Min이 공부하면 좋은 기술 포인트

### [Study] 1. Knowledge Distillation에서 KL과 CE의 역할

이번 recipe는 단순 CE 학습이 아니다.

```text
KL: teacher의 soft decision boundary를 따라가게 함
CE: hard label boundary를 조금 anchor함
```

`ce0.1_plus_teacher_kl`의 의미:

```text
Teacher-S의 ROI-derived disease boundary를 주로 따르되,
진짜 CN/AD label을 10% 정도만 anchor로 섞는다.
```

이게 KDRC에서 효과적이었던 이유는, 순수 KL만으로는 fold-specific boundary가 약하고, pure CE는 ROI-aware representation을 망칠 수 있기 때문이다.

### [Study] 2. Teacher ceiling과 Student failure를 분리해야 함

NACC에서 중요한 점:

```text
Teacher-S AUC가 0.8015로 낮다.
Student frozen AUC가 0.7597이다.
```

이 경우 학생만 탓하면 안 된다.

질문을 나눠야 한다.

```text
1. Teacher가 충분히 강한가?
2. Student가 teacher를 잘 모방했는가?
3. Student embedding이 downstream linear probe에서 transfer되는가?
4. held-out cohort의 label/ROI/QC distribution이 다른가?
```

### [Study] 3. LOCO 평가의 의미

LOCO는 leave-one-cohort-out이다.

```text
train: 나머지 cohort
test: held-out cohort 하나
```

이 평가는 random split보다 훨씬 어렵다.
왜냐하면 모델이 scanner/site/cohort shortcut을 쓰기 어렵고, 진짜 cross-cohort generalization을 봐야 하기 때문이다.

### [Study] 4. 평균 성능만 보면 안 됨

Gate04 평균은 pass다.

```text
mean frozen AUC: 0.8159
```

하지만 fold별로 보면:

```text
KDRC: 성공
NACC: 실패
AJU: 약한 실패
```

의료영상 연구에서는 평균 하나로 claim하면 위험하다. 반드시 fold-wise, cohort-wise, class-wise 결과를 같이 봐야 한다.

### [Study] 5. Teacher-B는 “정답”이 아니라 진단 도구

Teacher-B가 좋아졌다고 해서 바로 age/sex를 main model input에 넣자는 뜻은 아니다.

Teacher-B는 다음 질문을 위한 도구다.

```text
NACC에서 ROI-only signal이 약한 이유가 demographic/context axis 부족 때문인가?
```

이 질문에 답하기 위한 diagnostic teacher이지, 최종 VLM pretraining recipe로 자동 승격되는 것은 아니다.
