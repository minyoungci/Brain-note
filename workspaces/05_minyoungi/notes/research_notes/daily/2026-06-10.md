### 4. 전처리 코드 구조 정비 (preprocessing/)

```
preprocessing/
  ── dicom_to_nifti/          역할: DICOM → NIfTI
  │     aju.py                AJU: 3D_T1/T2_FLAIR/T2_FSE/DTI/PET, fMRI 제외
  │     adni.py               ADNI T1: adni_t1w_dicom_list.csv 기반, 동일날짜 최다슬라이스 우선
  │     nacc.py               NACC: zip 압축해제 → series명으로 modality 자동분류
  │     README.md
  │
  ── raw_manifest/             역할: manifest에 raw_*_path 컬럼 부착
  │     build.py               --dry-run / --verify 옵션
  │     resolvers/
  │       kdrc.py              glob "{subj}*/{subj}_1_{MOD}.nii.gz" (Korean suffix 있는/없는 dir 모두)
  │       a4.py                "A4_MR_{MOD}_{bid}_{viscode_num}.nii.gz" (A4_A4_ prefix 제거)
  │       oasis.py             session_dir/anat* + 별도 dwi_bids_v7_exact_dwi_only/ tree
  │       aibl.py              preprocessed_official/v2/AIBL/.../native_t1w_hdbet.nii.gz
  │       aju.py               nifti/{site}/{subj}/{visit}/{mod}/ (변환 후)
  │       adni.py              nifti/{subj}/{image_id}/t1/ (변환 후)
  │       nacc.py              nifti/{nacc_id}/{image_id}/{mod}/ (변환 후)
  │     README.md
```

**디버그 이슈 3개 발견·수정:**
1. KDRC: `{subj}_*` → `{subj}*` (suffix 없는 dir도 매치)
2. A4: `subject_id = "A4_A4_B10081264"` → BID = `split("_")[-1]`
3. OASIS DWI: session_dir 내부 아님 → `dwi_bids_v7_exact_dwi_only/sub-{subj}/ses-{sess}/dwi/`
