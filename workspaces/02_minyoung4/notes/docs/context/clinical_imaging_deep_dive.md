## Interpretation

- Tier 1 is sufficient for an all-dataset structural MRI baseline with age/sex/scanner covariates.
- Tier 2 supports the first supervised labels, especially IDH; conflict-excluded counts should be used for modeling.
- Tier 3 variables are valuable but require cohort restriction or semantic harmonization.
- Tier 4 should be treated as subset-specific rather than all-consortium common data.
- No split should be created by unit row; split by dataset + subject_id and report scanner/site balance.
