## Recommended Sequencing

1. Use Tier 1 variables and structural MRI core to define a common baseline protocol.
2. Run full NIfTI header audit only after approval.
3. Start supervised modeling with T1 structural IDH prediction after split/preprocessing approval.
4. Add segmentation-aware IDH as a variant after zero-byte/provenance handling.
5. Treat MGMT, survival/PFS, advanced imaging, and histopath as later or subset-specific tasks.
